# 🇹🇭 Lexicon TH — ภาษาไทย (Tailandés)

> Filosofía aplicada: el tailandés tiene un sistema tonal de 5 tonos y onomatopeyas (คำเลียนเสียง, kham lian siang) muy expresivas. La escritura tailandesa es un abugida con script propio. Se usó script nativo para todas las entradas con romanización referencial donde ayuda a la comprensión del sonido.
