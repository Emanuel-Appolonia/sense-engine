# 📖 Lexicon Master — SenseEngine

Este documento sirve como referencia centralizada para todas las traducciones del motor. Se utiliza para validar la consistencia entre idiomas y como base para futuras expansiones.

> [!IMPORTANT]
> **Filosofía de Traducción: Sensación sobre Literalidad**
> SenseEngine prioriza la traducción de **sensaciones y onomatopeyas** por sobre el significado literal de las palabras. Entendemos la complejidad intrínseca de cada sistema de escritura: desde los logogramas chinos (Hanzi) y los silabarios japoneses (Katakana para sonidos) hasta los abjads árabes, abugidas índicos (Devanagari/Bengali) y el alfabeto cirílico. 
> 
> En muchos casos, lo que para un idioma es una palabra descriptiva, para otro es un sonido abstracto (ej: en japonés se prefiere `ボウ` para el sonido del fuego en lugar de `火`). Este Lexicon busca el equilibrio entre la precisión técnica y la autenticidad orgánica del sonido en cada cultura.

> [!NOTE]
> **Aviso sobre Inconsistencias:**
> SenseEngine es un proyecto en constante evolución. Si bien se ha buscado la máxima precisión orgánica, reconocemos que en un ecosistema de 11 idiomas con sistemas de escritura tan diversos, pueden existir fallas lingüísticas, ortográficas o de contexto. Cualquier error detectado no ha sido intencionado y es responsabilidad del proceso de desarrollo actual. Se agradece el feedback para seguir puliendo la herramienta.
